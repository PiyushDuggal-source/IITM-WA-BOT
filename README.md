# IITM-WA-BOT

### This bot is for helping IITM Students to make their life easy and fast by providing the following helps:

- Reminds about upcoming classes/events
- Provides upcoming calendar events üìÖ in the WhatsApp itself
- Provides today's lectures
- Provides notes just by typing a single command
  - you can also use it individually for getting all notes or you can use the filter
    E.G: **!notes** Stats
- Provides course playlist
  - you can also use it individually for getting all playlist or you can use the filter
    E.G: **!playlist** Stats

## üó£ Commands

| Command               | Description                                       |
| --------------------- | ------------------------------------------------- |
| `!class`              | ‚úÖ Get the classes for the day                    |
| `!calendar`           | ‚úÖ Get the clalendar events/ upcoming Classes     |
| `everyone`‚≠ê          | ‚úÖ Ping everyone in a group                       |
| `!help`               | ‚úÖ Get more info                                  |
| `!allcmd`             | ‚úÖ Get all commands                               |
| `!notes`              | ‚úÖ Get all available notes (pdf, links etc.)      |
| `!notes <subName>`    | ‚úÖ Get specific subject's notes (pdf, links etc.) |
| `!playlist`           | ‚úÖ Get all course's playlists (links)             |
| `!playlist <subName>` | ‚úÖ Get specific subject's playlist (links)        |
| `!source`             | ‚úÖ Get the bot's source code                      |
| `!grouplink`          | Get the current group's invite link               |
| `!promote`‚≠ê          | Promote a user to be a bot admin                  |

‚≠ê = Commands that can only be used when you are a **bot Admin**.
